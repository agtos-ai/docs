# What is agtOS?

agtOS is an open-source, voice-native AI agent platform. It provides the infrastructure layer for AI agents that can hear, speak, think, and act — connecting voice input/output with LLM reasoning and tool execution.

## Architecture

agtOS uses a dual-layer architecture:

- **Infrastructure Layer**: Voice pipeline, audio processing, WebSocket transport
- **Orchestration Layer**: AI-driven via protocol gateway (MCP + A2A ready)

```
User Voice → STT (sherpa-onnx) → Model Router → Claude/Ollama → TTS → Speaker
                                    ↓
                              Agent Loop → Tool Registry → MCP Tools
                                    ↓
                           Memory (Working + Episodic + Semantic)
```
