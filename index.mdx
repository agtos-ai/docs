## Key Features

- **Voice Pipeline** — STT → LLM → TTS with WebSocket streaming, barge-in, and word-level timestamps
- **Multi-Provider AI** — Claude, Ollama, sherpa-onnx, OpenRouter — with three-tier model routing
- **MCP Integration** — 10 built-in tools + external server connections + dynamic tool selection
- **Memory System** — Working, episodic, and semantic tiers with Dialectic reasoning
- **Device Registry** — ESP32, browser, and CLI device management with per-device preferences
- **Web Dashboard** — Real-time health, voice, memory, devices, settings, and logs
- **Desktop App** — Tauri 2 with system tray, global push-to-talk hotkey, and auto-start
- **CLI Tools** — `agtos start`, `agtos setup`, `agtos chat`, `agtos doctor`, and more
