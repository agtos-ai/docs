## Key Features

- **Voice Pipeline**: STT → LLM → TTS with WebSocket streaming
- **Multi-Provider AI**: Claude, Ollama, speaches — with automatic routing
- **MCP Integration**: 9 built-in tools + external server connections
- **Memory System**: Working, episodic, and semantic tiers
- **Desktop App**: Tauri 2 with system tray and global hotkey
- **CLI Tools**: `agtos start`, `agtos setup`, `agtos chat`
