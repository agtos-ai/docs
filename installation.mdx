## Requirements

| Component | Version | Required |
|-----------|---------|----------|
| Node.js | 22+ | Yes |
| Docker | Latest | For voice (STT/TTS) |
| Redis | 7.2+ | For memory, scheduling |
| Ollama | Latest | For local AI models |

## Install Methods

### npm (Development)

```bash
git clone https://github.com/agtos-ai/agtos.git
cd agtos
npm install
```

### CLI (Global)

```bash
npm install -g agtos
agtos setup    # Interactive configuration
agtos start    # Start the server
```

The setup wizard validates API keys against the actual provider API and supports both:
- **API key** (`sk-ant-api03-...`) — pay-per-token from console.anthropic.com
- **Max subscription** (`sk-ant-oat01-...`) — flat-rate Claude subscription

Run `agtos doctor` to check system dependencies and credential status.

### Docker

```bash
docker compose up -d
```

### Desktop App

Download from [GitHub Releases](https://github.com/agtos-ai/agtos/releases).
