## Prerequisites

- Node.js 22+
- (Optional) Docker for voice services
- (Optional) Claude API key or Max subscription

## 1. Install

```bash
git clone https://github.com/agtos-ai/agtos.git
cd agtos
npm install
```

## 2. Configure Authentication

<Tabs>
  <Tab title="API Key (pay-per-token)">
    ```bash
    echo "ANTHROPIC_API_KEY=sk-ant-api03-your-key" >> .env.local
    ```
    Get your key from [console.anthropic.com](https://console.anthropic.com)
  </Tab>
  <Tab title="Max Subscription (flat rate)">
    ```bash
    claude setup-token
    # Copy the token, then:
    echo "ANTHROPIC_AUTH_TOKEN=sk-ant-oat01-your-token" >> .env.local
    ```
    agtOS spawns the `claude` CLI as a subprocess — no external proxy needed.
  </Tab>
</Tabs>

## 3. Start the Server

```bash
npx agtos setup   # First-run: validates API keys against provider API
npx agtos start   # Starts server with progress feedback
```

The dashboard and health endpoint are available immediately at `http://localhost:4102/health`.

## 4. Test Chat

```bash
curl -X POST http://localhost:4102/api/chat \
  -H 'Content-Type: application/json' \
  -d '{"text": "Hello, what is 2+2?"}'
```

## 5. (Optional) Start Voice Services

```bash
docker compose up -d redis
```

Then open the dashboard at `http://localhost:4102` and try voice chat.
