## Prerequisites

- **Node.js 22+** (required)
- **Docker** (optional, for Redis — enables memory, scheduling, and device management)
- **Ollama** (optional, enables local AI routing and semantic search)
- **Claude API key or Max subscription** (optional, for cloud AI)

## 1. Install

```bash
git clone https://github.com/agtos-ai/agtos.git
cd agtos
npm install
```

## 2. Configure Authentication

<Tabs>
  <Tab title="API Key (pay-per-token)">
    ```bash
    echo "ANTHROPIC_API_KEY=sk-ant-api03-your-key" >> .env.local
    ```
    Get your key from [console.anthropic.com](https://console.anthropic.com)
  </Tab>
  <Tab title="Max Subscription (flat rate)">
    ```bash
    claude setup-token
    # Copy the token, then:
    echo "ANTHROPIC_AUTH_TOKEN=sk-ant-oat01-your-token" >> .env.local
    ```
    agtOS spawns the `claude` CLI as a subprocess — no external proxy needed.
  </Tab>
</Tabs>

## 3. Start the Server

```bash
npx agtos setup   # First-run: validates API keys against provider API
npx agtos start   # Starts server with progress feedback
```

The dashboard and health endpoint are available immediately at `http://localhost:4102/health`.

## 4. Test Chat

<Tabs>
  <Tab title="CLI">
    ```bash
    npx agtos chat
    # Type your messages in the interactive prompt
    ```
  </Tab>
  <Tab title="curl">
    ```bash
    curl -X POST http://localhost:4102/api/chat \
      -H 'Content-Type: application/json' \
      -d '{"text": "Hello, what is 2+2?"}'
    ```
  </Tab>
</Tabs>

## 5. (Optional) Enable Memory and Scheduling

```bash
docker compose up -d redis
```

With Redis running, you get cross-session memory, task scheduling, device registry, and user preferences.

## 6. Open the Dashboard

Navigate to `http://localhost:4102` for the web dashboard — real-time health monitoring, voice interaction, memory browsing, and system configuration.

## Next Steps

- [Install Ollama](https://ollama.ai) for local AI routing and semantic memory search
- Run `npx agtos doctor` to verify all dependencies
- Download voice models: `npx agtos models download --default`
- See the [CLI Reference](/features/cli) for all available commands
