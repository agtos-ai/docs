Get from zero to a working agtOS instance in 5 minutes.

## Prerequisites

- **Node.js 22+** ([download](https://nodejs.org))
- **A Claude API key** or **Max subscription** (optional — enables cloud AI)

<Steps>
  <Step title="Clone and install">
    ```bash
    git clone https://github.com/agtos-ai/agtos.git
    cd agtos
    npm install
    ```
  </Step>

  <Step title="Run the setup wizard">
    The wizard checks dependencies, prompts for API keys, and validates them against the actual provider API:

    ```bash
    npx agtos setup
    ```

    <Tabs>
      <Tab title="API Key (pay-per-token)">
        Enter your `sk-ant-api03-...` key from [console.anthropic.com](https://console.anthropic.com).
      </Tab>
      <Tab title="Max Subscription (flat rate)">
        Run `claude setup-token`, then enter the `sk-ant-oat01-...` token.
      </Tab>
      <Tab title="No API key">
        Press Enter to skip. agtOS still works for local-only use with Ollama.
      </Tab>
    </Tabs>
  </Step>

  <Step title="Download voice models">
    Download the default speech models for the in-process voice engine (~460MB):

    ```bash
    npx agtos models download --default
    ```

    <Info>
    This step is optional. Without models, agtOS runs in text-only mode. Voice features activate automatically when models are available.
    </Info>
  </Step>

  <Step title="Start the server">
    ```bash
    npx agtos start
    ```

    You'll see startup progress for each service:

    ```
      ✓ Health endpoint
      ✓ Voice pipeline
      ✓ MCP server
      ✓ Task scheduler
      ✓ API routes

    agtOS server is running!
      Dashboard:  http://localhost:4102
      Health:     http://localhost:4102/health
      Voice WS:   ws://localhost:3000/audio
      MCP:        http://localhost:4100/mcp
    ```
  </Step>

  <Step title="Try it out">
    <Tabs>
      <Tab title="CLI chat">
        ```bash
        npx agtos chat
        ```
        Type messages in the interactive prompt. Use `/help` for commands, `/quit` to exit.
      </Tab>
      <Tab title="curl">
        ```bash
        curl -X POST http://localhost:4102/api/chat \
          -H "Content-Type: application/json" \
          -d '{"text": "Hello, what can you do?"}'
        ```
      </Tab>
      <Tab title="Dashboard">
        Open `http://localhost:4102` in your browser for the web dashboard with voice interaction, memory browsing, and system management.
      </Tab>
      <Tab title="Claude Desktop">
        Add agtOS as an MCP server in `claude_desktop_config.json`:

        ```json
        {
          "mcpServers": {
            "agtos": { "url": "http://localhost:4100/mcp" }
          }
        }
        ```
      </Tab>
    </Tabs>
  </Step>
</Steps>

## Optional: Enable memory and scheduling

Start Redis for cross-session memory, task scheduling, device registry, and user preferences:

```bash
docker compose up -d redis
```

<Tip>
No Docker? Run `npx agtos doctor` to check all dependencies and get remediation hints for anything missing.
</Tip>
